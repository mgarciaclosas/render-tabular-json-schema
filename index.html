JSON Schema → Data Dictionary

Upload one or more JSON Schema files to generate a searchable, exportable data dictionary table. All processing happens locally in your browser — no data is uploaded anywhere.

How to use

  1. Click Choose JSON Schema File(s) and select one or more .json schema files
  2. Click Generate Table to render the data dictionary
  3. Use the column selector to show, hide, or reorder columns
  4. Use the search box to filter variables by name or description
  5. Click Export to Excel to download the full dictionary as .xlsx

Supported schema structures

Single file Domain schema (type: object) Upload one schema file directly — e.g. Pregnancies.json. No wrapper needed.
Multi-file Multiple domain schemas Upload several type:object schemas together — all variables are combined into one table, with each schema shown as a separate category section.
Multi-file Dataset + row schema (allOf / $ref) Upload a dataset-level schema (type: array) with its row schema (and any referenced domain schemas). $ref links resolve automatically by $id or filename.
